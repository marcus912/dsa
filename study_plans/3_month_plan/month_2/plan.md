# Month 2: Core Structures & Search

**Goal:** Move beyond linear structures. Master hierarchical data and efficient searching techniques, which are frequently evaluated in technical reviews.

## Week 5: Stacks & Queues (incl. Monotonic Stack)
Understand LIFO/FIFO and use stacks to keep track of previous states.
*   Valid Parentheses (ID: 20)
*   Min Stack (ID: 155)
*   Evaluate Reverse Polish Notation (ID: 150)
*   Daily Temperatures (ID: 739) - *Monotonic Stack*
*   Largest Rectangle in Histogram (ID: 84) - *Hard*

## Week 6: Trees & Tries
Master recursion (DFS) and level-order traversal (BFS). Understand when to use a Trie for string prefix problems.
*   Maximum Depth of Binary Tree (ID: 104)
*   Invert Binary Tree (ID: 226)
*   Lowest Common Ancestor of a Binary Search Tree (ID: 235)
*   Binary Tree Level Order Traversal (ID: 102)
*   Implement Trie (Prefix Tree) (ID: 208)
*   Serialize and Deserialize Binary Tree (ID: 297) - *Hard*

## Week 7: Heaps / Priority Queues
Learn to efficiently track the top $K$ elements in a dataset.
*   Kth Largest Element in an Array (ID: 215)
*   Top K Frequent Elements (ID: 347)
*   Merge K Sorted Lists (ID: 23) - *Hard*
*   Find Median from Data Stream (ID: 295) - *Hard*

## Week 8: Binary Search (Standard & Solution Space)
Don't just search arrays; learn to binary search the *answer* space (e.g., "What is the minimum speed needed?").
*   Binary Search (ID: 704)
*   Search in Rotated Sorted Array (ID: 33)
*   Find Minimum in Rotated Sorted Array (ID: 153)
*   Koko Eating Bananas (ID: 875)
*   Median of Two Sorted Arrays (ID: 4) - *Hard*
